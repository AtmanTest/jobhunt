Comment ça marche

JobHunt scrape automatiquement les offres d'emploi QA/SDET depuis RemoteOK et bientôt d'autres sources. Les offres sont stockées localement dans une base SQLite sur ton Mac.

Flux de travail

  1. 1 Lance Rafraîchir pour importer les nouvelles offres
  2. 2 Parcours les offres filtrées (QA uniquement, non postulées)
  3. 3 Clique Cover pour générer une cover letter personnalisée
  4. 4 Clique Apply pour ouvrir le site et postuler manuellement
  5. 5 Marque comme postulé dans l'app pour tracker tes candidatures

Sources d'emploi

  • ✓ RemoteOK — API JSON, ~200 offres QA
  • ⚠ We Work Remotely — nécessite scraping HTML
  • ⚠ Malt, LinkedIn, Indeed — bloqués par Cloudflare

Commandes utiles

cd ~/Desktop/jobhunt && python3 app.py
→ Lance le dashboard sur http://localhost:5050

python3 scraper.py
→ Scrape les offres sans lancer le serveur
